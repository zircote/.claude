# HMH Content Templates

Templates and examples for creating on-brand HMH content.

## Presentations

### Title Slide
- HMH logo (top left or bottom right)
- Title in Micro typeface
- Use gradient background (pink-to-orange) or warm photography
- Tagline: "Growth in Motion" (optional)

### Content Slides
- White or light neutral background
- Single key message per slide
- Photography or illustration to support message
- Minimal text; use speaker notes for detail

### Closing Slide
- Call-to-action
- Contact information
- HMH logo
- Website: hmhco.com

## Documents

### Report/Whitepaper Structure
```
[HMH Logo - header]

[Title - Micro typeface, sentence case]
[Subtitle - supporting context]

Executive Summary
- 2-3 paragraph overview
- Key findings/takeaways

[Body Sections]
- Clear heading hierarchy
- Evidence-based claims
- Educator/student quotes
- Data visualizations

Conclusion
- Summary of key points
- Clear next steps/CTA

[HMH Logo - footer]
Contact: hmhco.com
```

### Email Templates

```
[Proof point or testimonial]

```

**Internal Communications:**
```
Subject: [Clear topic]

Team,

[Context/background]

[Key information - bulleted if multiple points]

[Next steps/action items]

[Sign-off]
```

## Marketing Copy

### Headlines
- Lead with outcome/benefit
- Use active verbs
- Sentence case
- Examples:
  - "Create growth for every student"
  - "Learning that adapts as students progress"
  - "Empower educators with insights that matter"

### Body Copy Principles
1. Open with educator/student benefit
2. Acknowledge the challenge
3. Present HMH solution
4. Include proof point
5. Clear call-to-action

### Product Descriptions

**Format:**
```
[Product Name] [grade levels]

[One-sentence value proposition]

[2-3 key features/benefits]

[Evidence/proof point]

[CTA: Learn more at hmhco.com/[product]]
```

**Example:**
```
Into Reading K-6

A comprehensive literacy program designed to build confident readers.

- Systematic phonics and word study
- Diverse, engaging texts
- Built-in assessment and differentiation

90% of schools using Into Reading report improved student engagement.

Explore Into Reading: hmhco.com/into-reading
```

## Social Media

### LinkedIn/Professional
- Thought leadership focus
- Educator success stories
- Research and evidence
- Professional development tips
- 1-2 sentences + link

### General Social
- Vibrant imagery
- Concise messaging (under 280 characters)
- Hashtags: #GrowthInMotion #HMH #K12Education
- Student/teacher celebration
- Resource sharing

### Post Structure
```
[Hook - benefit or question]
[Brief context]
[CTA or link]
[Hashtags]
```

## Web Content

### Page Structure
- Clear H1 (one per page)
- Scannable subheadings (H2, H3)
- Short paragraphs (2-3 sentences)
- Bulleted lists for features
- CTAs throughout
- Mobile-optimized

### Landing Page Formula
1. Hero: Headline + supporting image/video
2. Problem: What educators face
3. Solution: How HMH helps
4. Proof: Statistics, testimonials
5. Features: Key capabilities
6. CTA: Next step (demo, contact, download)

## Video/Multimedia

### Video Guidelines
- Open with human story (educator/student)
- Keep under 2 minutes for social
- Caption all content
- Brand colors in graphics
- Logo at end
- Music: Upbeat, inspirational, modern

### Thumbnail Best Practices
- Human face when possible
- Bright, warm colors
- Clear, readable text
- Avoid clutter
